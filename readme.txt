AestheticaVite

AestheticaVite - это веб-приложение, предназначенное для косметологической клиники. Оно предоставляет функциональные возможности, такие как регистрация пользователей, авторизация, запись на услуги, форму обратной связи и административная панель для управления базой данных. Этот README содержит обзор проекта, его структуру и инструкции по настройке и запуску приложения.

Особенности

    Регистрация пользователей: Позволяет новым пользователям создавать аккаунты.
    Авторизация: Аутентифицирует пользователей и предоставляет доступ к защищенным разделам сайта.
    Запись на услуги: Позволяет пользователям записываться на различные процедуры.
    Форма обратной связи: Собирает отзывы пользователей и отправляет их на сервер.
    Административная панель: Позволяет администраторам просматривать и редактировать записи в базе данных. (Добавление новых записей в таблицу в административной панели не реализовано)

Используемые технологии

    Frontend:
        HTML, CSS, JavaScript
        SCSS для стилизации
        Vite для разработки и сборки проекта
    Backend:
        PHP для серверной логики
        MySQL для управления базой данных
    Безопасность:
        Google reCAPTCHA для предотвращения автоматических отправок

Установка

Чтобы установить проект локально, выполните следующие шаги:

    Клонируйте репозиторий:
    bash
git clone https://github.com/yourusername/AestheticaVite.git
cd AestheticaVite

Установите зависимости:
bash
npm install

Настройте сервер:

    Убедитесь, что у вас установлен PHP и MySQL.
    Создайте базу данных и импортируйте структуру и данные 

Запустите приложение:

bash
    npm run dev

Использование

    Регистрация и авторизация:
        Перейдите на страницу регистрации, заполните необходимые поля и нажмите "Зарегистрироваться".
        Для входа используйте страницу авторизации.

    Запись на услуги:
        Выберите услугу и дату, после чего нажмите "Записаться".

    Обратная связь:
        Заполните форму обратной связи и отправьте свои данные.

    Административная панель:
        Просматривайте и редактируйте записи в базе данных.

Структура проекта

    src/ - Исходный код фронтенда
    css/ - Стили
    img/ - Изображения
    php/ - Серверная логика на PHP
    database.sql - Файл для создания структуры базы данных и начальных данных

API эндпоинты

    GET /getTables.php - Получить список таблиц
    GET /getFields.php - Получить поля таблицы
    GET /getRecords.php - Получить записи таблицы
    POST /updateRecord.php - Обновить запись
    POST /login.php - Авторизация пользователя
    POST /submitFeedback.php - Отправка отзыва
    POST /submitApplication.php - Отправка заявки на услугу

Тестирование

Тестирование проводилось для проверки всех основных функций приложения. Функционал реализован в полном объеме за исключением добавления новых записей в таблицы административной панели. Пользователи получают соответствующие уведомления при некорректном вводе данных.
